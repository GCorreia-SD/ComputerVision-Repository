# ComputerVision-Repository

This directory covers some of the projects developed during the Computer Vision course, covering the concepts of: 

- Color Histograms
- Edges
- Monadic Processes
- Dyadic Operations
- Kernels
- Image Color Treatment
- Color Grading
- Image Moment and Centroid
